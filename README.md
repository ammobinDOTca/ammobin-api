### how to run
```docker run ammobindotca/ammobin-api -p 8080:8080```

### dev stuff
1. add
```
127.0.0.1 redis
127.0.0.1 influx
```
to /etc/hosts

2. ```docker-compose run -d```
3. ```npm run dev```
4. ```npm run dev-worker```
